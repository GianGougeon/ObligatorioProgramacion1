Nosotros - PelisFlix

PelisFlix es una plataforma web desarrollada por Santiago Pino y Gianfranco Gougeon. Este proyecto se creó como parte de un trabajo obligatorio para la materia Programación 1, en el CTC Instituto Tecnológico de Rosario.

El objetivo principal fue desarrollar un sitio funcional y atractivo que permita a los usuarios alquilar películas de forma sencilla y rápida. A través de esta plataforma, hemos aplicado y combinado nuestros conocimientos de HTML, CSS y JavaScript para ofrecer una experiencia de usuario clara y moderna.

PelisFlix refleja nuestro compromiso con el aprendizaje continuo, la tecnología y el desarrollo web. Además, nos ha brindado la oportunidad de explorar nuestra creatividad y mejorar nuestras habilidades prácticas en programación.
